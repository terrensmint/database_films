# Лабораторная №5 - Управление базой данных фильмов

## Описание
База данных хранится в бинарном файле. Пример базы данных "example.bin" с записями некоторых фильмов стоит по умолчанию. Для работы программы были использованы умные массивы.

### Команды
  1) Импорт базы данных - позволяет загрузить пользовательский файл, который изначально пуст или предварительно был сохранен с помощью этой программы.
  2) Сохранение базы данных - позволяет экспортировать базу данных в новый файл, перезаписать существующий или текущий файл.
  3) Вывод базы данных - выводит все записи в консоль.
  4) Добавить запись - пользователь заполняет форму записи по полям, после чего запись добавляется в конец базы данных.
  5) Удалить запись - находится запись по введенному ID и удаляется из базы данных.
  6) Найти запись - позволяет найти запись по конкретному ID или найти запись по нескольким полям сразу.
  7) Редактировать запись - находится запись по введенному ID и пользователь редактирует необходимые ему поля.

## Установка и запуск
```bash
# Клонировать репозиторий
git clone https://github.com/terrensmint/database_films.git
# Перейти в папку проекта
cd database_films

# Запуск Makefile
make

# Запуск программы
./app
```
